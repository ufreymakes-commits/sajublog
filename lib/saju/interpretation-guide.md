# 사주 해석 로직 분석 및 레포트 작성 가이드

## 1. 추출하는 핵심 데이터

### 1.1 기본 정보
- **생년월일시** (양력 기준)
- **성별**
- **관계** (본인/배우자 등)

### 1.2 사주 원국 데이터
- **천간(天干)**: 시주-일주-월주-연주 각각의 천간 (갑, 을, 병, 정...)
- **지지(地支)**: 시주-일주-월주-연주 각각의 지지 (자, 축, 인, 묘...)
- **십성(十星)**: 각 천간과 지지에 대한 십성 (비견, 겁재, 식신, 상관, 편재, 정재, 편관, 정관, 편인, 정인)
- **지장간(支藏干)**: 각 지지 속에 숨어있는 천간들
- **십이운성(十二運星)**: 각 기둥별 생명력 상태 (장생, 목욕, 관대, 건록, 제왕, 쇠, 병, 사, 묘, 절, 태, 양)
- **신살(神殺)**: 천을귀인, 태극귀인, 현침살, 도화살, 역마살, 귀문관살, 원진살 등 각종 신살
- **형충파해합**: 지지 간의 관계 (충, 합, 형, 파, 해, 원진, 반합 등)
- **일간 강약**: 신강(身强) 또는 신약(身弱) 판단
- **격국(格局)**: 사주의 전체 구조 패턴 (식상생재격, 재생관격, 인수격 등)
- **용신(用神)**: 사주에 필요한 오행

### 1.3 오행 분석 데이터
- **오행 분포**: 목, 화, 토, 금, 수의 개수 및 강약
- **오행구족 여부**: 5가지 오행이 모두 있는지
- **무관/무재/무식/무인 여부**: 특정 십성이 없는 경우
- **과다 오행**: 특히 많은 오행
- **부족 오행**: 특히 적거나 없는 오행

### 1.4 일주 특성 데이터
- **일주의 간지 조합**: (예: 기미, 을축, 기사)
- **일지의 십성**: 배우자궁의 성격
- **일지의 지장간**: 내면에 숨겨진 기운
- **조후(調候)**: 계절적 특성 (뜨겁다, 차갑다, 습하다, 건조하다)

### 1.5 구조적 특징 데이터
- **합화(合化)**: 천간합이나 지지합으로 오행 변화
- **재다신약/재약신강**: 재성과 신(일간) 간의 힘의 균형
- **관왕재왕/관약재약**: 관성과 재성의 세력
- **통근(通根) 여부**: 천간이 지지에 뿌리를 내렸는지
- **간여지동**: 천간과 지지가 같은 오행일 때

## 2. 섹션별 데이터 활용 및 작성 방법론

### 섹션 1: 한 문장 정의 (대제목)

**사용 데이터:**
- 일주(일간+일지)
- 주변 천간/지지의 오행 분포
- 신살 전반 (ex. 귀인살, 도화살, 역마살, 귀문관살, 원진살 등)
- 형충파해합 관계

**작성 로직:**
1. **일주를 자연물에 비유** (예: 기미 = 사막의 흙, 을축 = 겨울 땅의 풀, 기사 = 뜨거운 용암 품은 밭)
2. **주변 오행을 환경으로 묘사** (예: 물이 많으면 "강물이 흐르는", 금이 많으면 "칼날이 도사린")
3. **긴장감/역동성 표현** (원진, 충, 귀문관살 등이 있으면 "아슬아슬한", "치열한")
4. **외유내강/반전 매력 강조** (일지 내부의 지장간과 겉모습의 대비)

**예시 패턴:**
- "물길을 다스리는 사막의 붉은 장미" (기미 + 수 과다)
- "폭풍우 치는 바위 틈에서 피어난 강철 장미" (을축 + 충/원진)
- "폭우 쏟아지는 비옥한 텃밭" (기사 + 수 과다)

**작성법:**
```
[일간의 성질] + [주변 환경 묘사] + [비유적 정체성]
```

### 섹션 2: 오행 분석

**사용 데이터:**
- 오행별 개수 (목/화/토/금/수)
- 과다/부족 오행
- 십성 결핍 여부 (무관/무재/무식/무인 등)
- 조후(調候) - 계절적 특성 (뜨거움/차가움/습함/건조함)

**작성 로직:**
1. **현재 오행 상태를 자연 풍경으로 묘사**
   - 물 많음 → "비가 억수같이 쏟아지는", "강물이 범람하는"
   - 금 많음 → "칼날이 많은", "쇠창고"
   - 목 없음 → "나무가 하나도 없는 평야"

2. **무엇이 문제인지 명확히 지적**
   - 목(관성) 부족 → "나를 얽매는 틀이 없어 자유롭지만 방향성 없음"
   - 수(재성) 과다 → "물이 많아 흙이 쓸려나갈 위험"

3. **오행 원리로 풀이**
   - "기토에게 목은 관성, 즉 직장/명예/남편을 의미"
   - "물은 재물이지만 너무 많으면 탐욕과 우울로 변질"

4. **개운법 제시**
   - 부족한 오행을 채우는 구체적 방법
   - 색깔, 방향, 습관, 취미, 인테리어

**예시 구조:**
```
1문단: 오행 분포 전체 그림 그리기 (비유)
2문단: 문제점/특이점 설명 (명리학 원리)
3문단: 실생활 영향 및 성격 연결
4문단: 개운법 (구체적 행동 지침)
```

### 섹션 3: 일주 분석

**사용 데이터:**
- 일주 간지 (60갑자 중 해당하는 것)
- 일지의 십성 (비견/겁재/식신/상관/편재/정재/편관/정관/편인/정인 중)
- 일지의 지장간 (숨어있는 천간들)
- 십이운성 (장생/목욕/관대/건록/제왕/쇠/병/사/묘/절/태/양 중)
- 일지와 다른 지지의 관계 (충/합/형/파/해/원진/반합 등)

**작성 로직:**
1. **일주의 이중성 강조**
   - 겉모습 vs 속마음
   - 천간(외부) vs 지지(내부) 대비
   - 예: "겉은 순박한 양, 속은 고집 센 황소"

2. **십성 해석**
   - 일지가 재성 → "현실 감각 탁월, 돈 냄새를 잘 맡음"
   - 일지가 인성 → "직관력과 센스가 비상"
   - 일지가 비겁 → "자립심 강하고 경쟁심 왕성"

3. **일지 내부의 지장간 분석**
   - 숨어있는 편관 → "억압받는 느낌, 스트레스"
   - 숨어있는 재성 → "실속형, 계산적"

4. **충/합/원진 영향**
   - 원진 → "예민함, 애증, 불안감"
   - 충 → "변동, 불안정, 역동성"
   - 합 → "인연, 융화, 변화"

5. **현실 조언**
   - 일주의 단점을 장점으로 승화시키는 방법
   - 피해야 할 것과 활용해야 할 것

**예시 구조:**
```
1문단: 일주의 이미지 비유
2문단: 십성과 십이운성 해석
3문단: 형충합/신살 영향
4문단: 현실적 조언
```

### 섹션 4: 단점/비판 (팩폭)

**사용 데이터:**
- 비견/겁재의 과다 여부
- 십성 결핍 (특히 무관)
- 형충파해합 중 부정적 작용 (충/형/원진 등)
- 재성과 일간의 힘 균형 (재다신약 등)
- 신살 중 부정적 요소 (ex. 도화살, 홍염살, 귀문관살 등)

**작성 로직:**
1. **직접적이고 솔직한 톤 사용**
   - "솔직히 말해서", "팩폭 한번 날리자면"
   - 2인칭 반말 사용 ("~하시죠?", "~있을 겁니다")

2. **독자 공감 유도**
   - 독자의 숨겨진 행동 패턴 지적
   - 예: "겉으로는 네네 하면서 속으로는 너나 잘하세요"

3. **명리학적 근거 제시**
   - "이는 ~의 기운 때문입니다"
   - 무관 → 고집, 자유분방, 무책임
   - 원진 → 예민함, 의심, 피해의식
   - 비겁과다 → 경쟁심, 고집, 자기중심

4. **따끔한 조언**
   - "~좀 내려놓으세요"
   - "~하지 않으면 고립될 것입니다"

**주요 비판 패턴:**
- 고집/독불장군 → 십성 결핍(특히 무관) 사주
- 예민함/뒤끝 → 형충파해합 중 부정적 관계 (원진/귀문/충 등)
- 충동성/무계획 → 재다신약 또는 식상 과다
- 인색함/속물근성 → 재성 과다
- 관종/허세 → 신살 중 화려함 관련 (ex. 도화/상관 등)

**예시 구조:**
```
1문단: 공감형 질문으로 시작
2문단: 명리학적 원인 설명
3문단: 결과적 문제점
4문단: 따끔한 조언
```

### 섹션 5: 장점/격려

**사용 데이터:**
- 신살 중 긍정적 요소 (ex. 천을귀인, 태극귀인, 학당귀인, 복성귀인 등)
- 일지의 십이운성 중 강한 것 (제왕/건록/관대 등)
- 강한 오행의 긍정적 해석
- 특별한 사주 구조 (ex. 등라계갑, 간여지동, 육합/삼합 등)

**작성 로직:**
1. **감성적 톤으로 전환**
   - "지금까지 얼마나 치열하게 버텨오셨나요?"
   - "당신은 ~한 능력을 지닌 분입니다"

2. **귀인살 강조**
   - "전생에 나라를 구했거나 조상님의 보살핌"
   - "위기의 순간마다 보이지 않는 손이 도와줌"

3. **단점의 재해석**
   - 예민함 → 섬세함, 통찰력
   - 고집 → 뚝심, 일관성
   - 물 많음 → 유연함, 재물운

4. **성장 스토리 구성**
   - "남들은 모르는 마음의 상처가 있었지만"
   - "그것을 자양분 삼아 더 큰 그릇으로 성장"

5. **자기긍정 메시지**
   - "당신은 이미 충분히 훌륭합니다"
   - "스스로를 더 믿고 사랑해 주세요"

**예시 구조:**
```
1문단: 고난에 대한 공감
2문단: 숨겨진 능력/귀인복 강조
3문단: 단점의 재해석
4문단: 격려 및 자기긍정 유도
```

### 섹션 6: 성격 분석

**사용 데이터:**
- 일간의 음양 및 오행 (갑을병정무기경신임계)
- 일지의 십성 및 지장간
- 신살 중 성격 관련 (ex. 도화살, 홍염살, 귀문관살, 원진살 등)
- 비견/겁재와 식상의 관계
- 형충파해합의 영향

**작성 로직:**
1. **성격을 자연 현상에 비유**
   - "한여름 소나기 같다"
   - "카멜레온 같다"
   - "소심한 관종"

2. **겉모습 vs 속마음 대비**
   - 일간(양/음) → 겉으로 보이는 성격
   - 일지 → 실제 내면
   - 예: "겉은 조용, 속은 열정"

3. **형충합의 영향**
   - 충 → "감정 기복이 롤러코스터"
   - 원진 → "애증의 감정, 변덕"
   - 합 → "융화성, 적응력"

4. **후천적 vs 선천적 성격**
   - 되고 싶은 성격 (이상)
   - 실제 성격 (현실)

5. **개선 방법**
   - 기록하는 습관
   - 거절 연습
   - 템포 조절

**예시 구조:**
```
1문단: 성격 비유
2문단: 형성 원인 (오행/신살)
3문단: 겉과 속의 괴리
4문단: 변화 방법
```

#### 7-1. 기본 적성 및 업무 스타일 (독자 친화적 설명)

**작성 로직:**
1. **조직 vs 프리랜서 판단**
   - 무관 → "조직 생활은 숨 막힘, 자영업/사업 추천"
   - 관성 강 → "전문직, 공무원, 대기업"

2. **큰 틀의 직업군 제시 (오행 기반)**
   - 재성 강 → 금융, 무역, 유통, 부동산
   - 인성+식상 → 기획, 디자인, IT, 의료, 교육
   - 신살 중 정밀함 관련 (ex. 현침살) → 정밀 기술, 의료, 금융 분석
   - 신살 중 화려함 관련 (ex. 도화살) + 식상 → 예술, 연예, 미디어

3. **잘하는 것 vs 못하는 것**
   - 디테일형 vs 큰그림형
   - 실무형 vs 기획형
   - 안정형 vs 도전형

**예시 구조 (독자용):**
```
1문단: 조직 적합도 + 업무 스타일
2문단: 오행/십성 기반 큰 틀의 직업군
3문단: 능력 분석 (강점/약점)
```

#### 7-2. 구체적 커리어 경로 분석 (체계적 심화)

**분석 단계 (내부 작업):**

**Step 1: 전체 에너지 구조 파악**
- **격국(格局) 식별**: 사주의 전체적 패턴 파악
  - 예: 식상생재격, 재생관격, 인수격, 상관배인격 등
- **일간 강약 판단**:
  - 신강(身强): 비견/겁재가 많거나, 인성이 강하거나, 월령을 얻음 → 자율성/리더십 필요
  - 신약(身弱): 재관식상이 많거나, 비인이 약하거나, 월령을 잃음 → 시스템/협업 필요
- **용신(用神) 파악**: 부족하거나 필요한 오행 확인
  - 예: 화(火) 격국이면 목(木, 화를 생성)과 토(土, 화가 제어)가 중요

**Step 2: 오행별 직업 분야 매핑**
- **목(木)**: 교육, 성장, 출판, 콘텐츠, 개발, 기획
- **화(火)**: 미디어, 예술, 공연, 영향력, 마케팅
- **토(土)**: 부동산, 경영, 시스템, 안전, 건축
- **금(金)**: 금융, 법률, 기술, 기계, 의료
- **수(水)**: 심리, 커뮤니케이션, 물류, 연구, 여행

**Step 3: 십성별 역할 매핑**
- **정관(正官)**: 공직, 행정, 법조, 질서 유지
- **편관(偏官)**: 보안, 국방, 군대, 단속
- **정재(正財)**: 비즈니스, 재무, 부동산, 안정적 수입
- **편재(偏財)**: 영업, 무역, 미디어, 빠른 거래
- **식신(食神)**: 요리, 스포츠, 공연, 여유로운 창작
- **상관(傷官)**: 예술, 미디어, 디자인, 표현, 반항
- **정인(正印)**: 교육, 치료, 상담, 전통
- **편인(偏印)**: 심리, 철학, 혁신, 연구
- **비견/겁재(比肩/劫財)**: 창업, 리더십, 그룹 조율

**Step 4: 財·官·傷·印·比 균형 분석**

십성 간 상호작용으로 커리어 프로필 생성:

| 십성 조합 | 직업 프로필 |
|---------|----------|
| **인성(印)과 비견(比) 강함** | 연구자, 교육자, 분석가 — 깊이 있는 학습과 독립적 사고 선호 |
| **재성(財), 관성(官), 인성(印) 균형** | 안정적 팀 리더, 전략가 — 기획력과 리더십이 뛰어나며 구조화된 환경에서 강점 |
| **식상(傷)과 재성(財) 강하고 관성(官) 약함** | 독립 크리에이터, 창업가 — 자유와 혁신 선호, 위계질서 거부 |
| **관성(官)과 식상(傷) 모두 강함** | 의료 커뮤니케이터, 대중 전문가 — 지식, 규율, 표현을 결합한 서비스 지향 역할 |
| **재성(財) 과다, 비견(比) 약함** | 재무 관리자 필요, 파트너십 — 혼자보다는 협업으로 돈 관리 |
| **식상(傷)만 강함** | 예술가, 작가, 디자이너 — 순수 창작, 표현 중심 |

**Step 5: 근무 형태 추천**
- **신강(身强)**: 창업가적 마인드, 리더십 역할, 자율성 선호
  - 추천: 프리랜서, 스타트업 창업자, 크리에이터
- **신약(身弱)**: 시스템 지원, 안정적 협업 선호
  - 추천: 정규직, 팀 기반 협업, 체계적 조직

### 섹션 8: 재물운

**사용 데이터:**
- 재성(편재/정재)의 개수와 위치
- 재다신약 vs 재약신강
- 비견/겁재의 유무 및 강약
- 일지의 재성 유무 (배우자 재물)
- 지장간의 재성

**작성 로직:**
1. **재물운의 구조 설명**
   - 재성 많음 → "기회가 널림"
   - 재성 적음 → "늦되게 모음"
   - 비겁+재성 → "벌어도 새어나감"

2. **돈 버는 방식**
   - 편재 → "한 방, 투자, 사업"
   - 정재 → "월급, 안정적"
   - 재다신약 → "흐르는 대로, 통제 필요"

3. **돈 쓰는 성향**
   - 지지 중 고정성 강한 것(ex. 축/술/진/미) → "저축형, 자린고비"
   - 신살 중 화려함 관련 (ex. 도화살) → "충동 소비"
   - 비견/겁재 강 → "베풂, 한턱 쏘기"

4. **재테크 방향**
   - 토 강 → "부동산"
   - 수 강 → "유동 자산 조심"
   - 금 강 → "금융 상품"

5. **배우자 재물**
   - 일지 재성 → "배우자가 경제적 능력"

**예시 구조:**
```
1문단: 재물운 규모
2문단: 돈 버는 방식
3문단: 문제점 (새는 것)
4문단: 재테크 조언
```

### 섹션 9-1: 연애운 및 매력

**사용 데이터:**
- 신살 중 매력 관련 (ex. 도화살, 홍염살)
- 식상(식신/상관)의 강약 - 표현력과 매력 발산
- 일간의 오행과 성격
- 형충파해합 중 사회성 관련

**작성 로직:**

1. **연애 매력 신호 분석**
   - 도화살/홍염살 → "이성에게 인기, 자연스러운 끌림"
   - 식상 강함 → "표현력 좋음, 재치 있음, 사교적"
   - 일간 오행별 매력 포인트:
     - 목(木): 성실하고 따뜻한 매력
     - 화(火): 밝고 열정적인 매력
     - 토(土): 편안하고 포근한 매력
     - 금(金): 세련되고 쿨한 매력
     - 수(水): 신비롭고 감성적인 매력

2. **사회적 인기도**
   - 비견/겁재 + 도화 → "친구들 사이에서 인기"
   - 식상 + 도화 → "SNS, 모임에서 주목받음"
   - 무관 + 도화 → "인기는 많지만 헌신은 별개"

3. **연애 스타일**
   - 신강 → "자유로운 연애, 쫓기는 것보다 쫓는 것 선호"
   - 신약 → "의지하고 싶은 연애, 안정감 추구"

4. **연애 타이밍 윈도우 (1-2개)**
   - 대운/세운 분석으로 연애 활성화 시기 파악
   - 구체적 연령대 또는 연도 제시
   - 예: "2025-2027년(만 28-30세), 인연이 많이 들어오는 시기"
   - 예: "30대 초반과 30대 후반, 두 차례 연애운 상승"

**예시 구조:**
```
1문단: 매력 신호 분석 (도화/식상)
2문단: 사회적 인기 및 연애 스타일
3문단: 연애 타이밍 (구체적 연령/연도)
```

### 섹션 9-2: 결혼운 및 배우자

**사용 데이터:**
- **배우자성(配偶星)**: 여성=관성(편관/정관), 남성=재성(편재/정재)
- 배우자성의 위치 (천간/지지 주기운/일지 지장간/기타 지장간/부재)
- 일지(배우자궁)의 십성
- 일지의 형충파해합 상태
- 일간의 강약 (신강/신약)
- 배우자성의 오행

**작성 로직:**

**Step 1: 배우자성 정의 및 가시성 분석 (필수)**

1. **배우자성 명확히 정의**
   - 여성: 관성(官星) - 나를 제어하고 이끄는 존재
   - 남성: 재성(財星) - 내가 관리하고 책임지는 존재

2. **배우자성 가시성 위계 (5단계)**
   
   **1순위: 천간에 배우자성 있음**
   - 해석: 배우자의 존재가 명확하고 직접적
   - 관계 경로: 일찍 정의되거나 명확해짐
   - 예: "당신의 배우자는 외부에서 명확히 드러나는 존재로, 연애 초기부터 결혼을 염두에 둔 관계로 발전하기 쉽습니다."

   **2순위: 지지 주기운에 배우자성 있음**
   - 해석: 배우자 인연이 탄탄하게 자리 잡음
   - 관계 경로: 안정적이고 지속 가능한 관계
   - 예: "배우자 인연은 내면 깊숙이 자리 잡고 있어, 한 번 맺어지면 쉽게 흔들리지 않습니다."

   **3순위: 일지 지장간(배우자궁 내부)에만 있음**
   - 해석: 배우자가 서서히 드러남, 경험을 통해 알아감
   - 관계 경로: 친구에서 연인으로, 연인에서 배우자로 천천히 발전
   - 예: "결혼 상대는 오랫동안 알고 지내던 사람이거나, 처음엔 몰랐다가 시간이 지나며 '이 사람이구나' 깨닫게 되는 경우가 많습니다."

   **4순위: 다른 지지 지장간에만 있음**
   - 해석: 배우자 인연이 약하게 숨어있음
   - 관계 경로: 타이밍과 운에 더 의존적
   - 예: "배우자 인연이 약하게 잠재되어 있어, 적극적으로 만남을 만들어야 합니다."

   **5순위: 원국에 배우자성이 완전 부재**
   - 해석: 결혼/헌신은 대운 활성화에 크게 의존
   - 주의: 원국만으로 결혼 가능 여부 단정 금지
   - 예: "원국에 배우자성이 없다는 것은 결혼을 못한다는 뜻이 아니라, 인생 특정 시기(대운)에 배우자 인연이 들어온다는 의미입니다."

**Step 2: 일간 강약에 따른 관계 태도 (필수)**

| 일간 강약 | 관계 태도 및 행동 패턴 |
|---------|-------------------|
| **신강(身强)** | • 관계에서 주도권을 쥐려는 성향<br>• 통제받거나 간섭받는 것 극도로 싫어함<br>• 상대방의 존중과 자율성 중시<br>• 배우자가 일관성 없거나 의존적이면 답답함<br>• 자기 주장이 강하고 독립적 |
| **신약(身弱)** | • 배우자의 지지와 안정감을 갈구<br>• 정서적 안전이 최우선<br>• 상대에게 적응하고 맞추려는 노력<br>• 경계가 불분명하면 에너지 소진<br>• 과도하게 수용하여 본인이 희생될 위험 |

**Step 3: 배우자궁(일지) 상태 분석**

배우자궁 = 결혼 생활의 그릇 (안정성 평가)

1. **합(合) - 결합, 애착, 파트너십 형성**
   - 긍정: 배우자와 강한 유대감, 서로를 필요로 함
   - 주의: 지나친 밀착은 숨 막힘, 의존 가능성
   - 예: "배우자와 끈끈한 관계를 맺지만, 각자의 공간도 필요합니다."

2. **충(沖) - 갈등, 분리, 변동**
   - 해석: 배우자와 물리적/정서적 거리 발생
   - 패턴: 주말부부, 원거리 연애, 잦은 이동
   - 예: "배우자와 같은 공간에 오래 있으면 부딪히기 쉽습니다. 적당한 거리가 오히려 관계를 건강하게 만듭니다."

3. **해(害)/원진(怨嗔) - 정서적 거리, 원망**
   - 해석: 신호를 잘못 읽음, 말 안 해도 알겠지 기대, 쌓이는 서운함
   - 패턴: 작은 오해가 누적되어 폭발
   - 예: "배우자와 소통이 엇갈리고 서운함이 쌓이기 쉽습니다. 말로 명확히 표현하는 것이 필수입니다."

4. **형(刑) - 내적 갈등, 자기 처벌**
   - 해석: 관계 안에서 자책하거나 과도한 책임감
   - 예: "배우자 문제를 내 탓으로 돌리는 경향이 있습니다. 경계를 분명히 하세요."

**Step 4: 배우자 오행 특성 및 타입**

배우자성의 오행으로 배우자의 에너지와 성향 파악

| 배우자성 오행 | 성향 및 특징 | 외모/분위기 |
|-----------|-----------|----------|
| **목(木)** | • 성장 지향적, 원칙적<br>• 배움과 발전 추구<br>• 고집 있지만 정직함 | • 키가 크거나 날씬한 체형<br>• 직선적이고 단정한 인상<br>• 맑고 깨끗한 느낌 |
| **화(火)** | • 열정적, 표현적<br>• 밝고 사교적<br>• 감정 기복 있을 수 있음 | • 밝은 인상, 화사한 분위기<br>• 또렷한 이목구비<br>• 패션 센스 있음 |
| **토(土)** | • 안정적, 신뢰감<br>• 묵묵하고 든든함<br>• 현실적, 실속 챙김 | • 중후한 체격, 포근한 인상<br>• 편안하고 온화한 느낌<br>• 무난하고 차분한 스타일 |
| **금(金)** | • 규율적, 구조적<br>• 냉철하고 논리적<br>• 완벽주의 성향 | • 날카롭고 세련된 인상<br>• 단정하고 깔끔한 스타일<br>• 차갑거나 도시적인 분위기 |
| **수(水)** | • 유연하고 적응적<br>• 감성적, 섬세함<br>• 변화에 능함 | • 부드럽고 물 흐르는 듯한 분위기<br>• 신비롭거나 몽환적 인상<br>• 곡선적이고 유려한 느낌 |

**참고**: 배우자성이 편(偏)이면 좀 더 자유롭고 파격적, 정(正)이면 정통적이고 안정적 성향 추가

**Step 5: 결혼 타이밍 분석 (필수)**

대운/세운 분석으로 결혼 최적기 1개 윈도우 제시

**타이밍 결정 로직:**
- 배우자성 활성화 (대운/세운에 배우자성 등장)
- AND 배우자궁 안정화/지지 (충/형 완화, 합/생 활성화)
- AND 용신 지원 (필요한 오행이 들어옴)

**출력 형식:**
- 구체적 연령대 + 연도 (예: "2028-2030년, 만 35-37세")
- 전문 용어(대운/세운/용신) 사용 금지
- 독자 친화적 표현 사용

**예시:**
```
"결혼 적기는 30대 중후반, 구체적으로는 2028-2030년(만 35-37세)입니다. 
이 시기에는 안정적인 인연이 들어오고, 배우자와의 궁합도 좋아져 
자연스럽게 결혼으로 이어질 가능성이 높습니다."
```

**Step 6: 주의사항 및 성장 조언 (사주 근거 명시)**

조언은 반드시 다음 중 하나 이상의 사주 데이터에 근거:

1. **불균형 패턴**
   - 예: 재성 과다 + 무관 → "돈에 집착하다 관계 놓칠 수 있음"
   
2. **배우자궁 상호작용**
   - 예: 일지 충 → "물리적 거리 유지 필요"
   - 예: 일지 원진 → "소통 명확히 하기"

3. **배우자성 가시성 문제**
   - 예: 배우자성 지장간에만 → "서두르지 말고 천천히 관계 발전시키기"
   - 예: 배우자성 부재 → "조급해하지 말고 타이밍 기다리기"

4. **일간 강약 불일치**
   - 예: 신강 + 신약 배우자성 → "상대 존중하되 본인 의견도 분명히"
   - 예: 신약 + 강한 관성 → "눌리지 않도록 자기 주장 키우기"

**예시 구조:**
```
1문단: 배우자성 정의 및 가시성 분석
2문단: 일간 강약에 따른 관계 태도
3문단: 배우자궁(일지) 상태 (합/충/해/원진)
4문단: 배우자 오행 특성 및 타입 (성향 + 외모)
5문단: 결혼 타이밍 (구체적 연령/연도)
6문단: 주의사항 및 성장 조언 (사주 근거 명시)
```

### 섹션 10: 가족 관계

**사용 데이터:**
- 연주/월주 (부모궁)
- 일지 (배우자궁)
- 시주 (자식궁)
- 인성(편인/정인) - 어머니
- 재성(편재/정재) - 아버지
- 비견/겁재 - 형제자매
- 식상(식신/상관) - 자식
- 신살 중 가족 관련 (ex. 효신살, 백호살 등)

**작성 로직:**
1. **부모와의 관계**
   - 인성 위치와 상태 → 어머니와의 관계
   - 재성 위치와 상태 → 아버지와의 관계
   - 연월의 충/합/원진 → 부모 간 관계, 본인과의 갈등

2. **부모 각각의 특징**
   - 인성이 충/약함 → "어머니와 애증 또는 거리감"
   - 재성이 신살 중 이동 관련 (ex. 역마살) → "아버지가 바쁘셨거나 떨어져 계심"

3. **형제자매 관계**
   - 비견/겁재 많음 → "경쟁 관계, 금전 거래 주의"

4. **자식복**
   - 식상(식신/상관)의 위치와 상태
   - 시주의 십성과 오행

5. **조언**
   - 독립의 필요성
   - 적당한 거리 유지
   - 금전 거래 금지

**예시 구조:**
```
1문단: 부모와의 전반적 관계
2문단: 어머니
3문단: 아버지
4문단: 형제/자식
5문단: 화목 방법
```

### 섹션 11: 인간관계/친구

**사용 데이터:**
- 비견/겁재의 유무 및 강약
- 쟁재 구조 (겁재+재성 구조)
- 신살 중 인간관계 관련 (ex. 귀문관살, 원진살, 도화살, 홍염살 등)
- 신살 중 이동 관련 (ex. 역마살 등)

**작성 로직:**
1. **친구 관계 스타일**
   - 비견/겁재 많음 → "친구 많지만 경쟁 관계"
   - 신살 중 고독 관련 (ex. 귀문관살/화개살) → "혼자 시간 필요"
   - 신살 중 매력 관련 (ex. 도화살) → "인기 많음"

2. **문제점**
   - 쟁재 → "친구가 내 돈 노림"
   - 겁재 → "배신, 뒤통수"
   - 원진 → "오해, 피로감"

3. **이상적 친구 vs 현실**
   - 좋은 친구: 부족한 오행 가진 사람
   - 나쁜 친구: 과다 오행 가진 사람

4. **조언**
   - 인맥 다이어트
   - 동업 금지
   - 손절의 중요성

**예시 구조:**
```
1문단: 친구 관계 현황
2문단: 문제점
3문단: 좋은 친구/나쁜 친구
4문단: 조언
```

### 섹션 12: 방향/장소

**사용 데이터:**
- 부족한 오행
- 과다한 오행
- 신살 중 이동 관련 (ex. 역마살, 지살, 천살 등)
- 조후 (뜨거움/차가움/습함/건조함)

**작성 로직:**
1. **길한 방향**
   - 부족한 오행의 방향
   - 목 → 동쪽
   - 화 → 남쪽
   - 금 → 서쪽
   - 수 → 북쪽

2. **국내 지역**
   - 수 부족 → 강/바다/호수 근처
   - 화 부족 → 남쪽/따뜻한 곳
   - 목 부족 → 산/숲 근처

3. **해외운**
   - 신살 중 이동 관련 (ex. 역마살) → "해외와 인연"
   - 신살 중 타향 관련 (ex. 지살/천살) → "고향 떠나야 발복"
   - 추천 국가 (방향과 오행 기준)

4. **주거 환경**
   - 부족한 오행 채우는 인테리어
   - 향(동서남북)
   - 지형 (물가, 산, 도심)

**예시 구조:**
```
1문단: 길한 방향
2문단: 국내 추천 지역
3문단: 해외운
4문단: 주거 팁
```

### 섹션 13: 마무리 격려

**사용 데이터:**
- 전체 사주의 핵심 메시지
- 개운 방향 (부족한 오행)
- 장점 재강조

**작성 로직:**
1. **사주의 핵심 비유 반복**
   - 첫 문단의 이미지 재소환

2. **투쟁과 성장 스토리**
   - "지금까지의 고난은 과정"
   - "더 큰 그릇이 되기 위한 준비"

3. **개운의 핵심 요약**
   - "~을 채우세요"
   - "~을 믿으세요"

4. **희망과 자신감 부여**
   - "당신은 이미 충분합니다"
   - "스스로를 믿으세요"

5. **구체적 행동 촉구**
   - "오늘 거울을 보며 말해주세요"
   - "이제 ~하세요"

**예시 구조:**
```
1문단: 핵심 이미지 재소환
2문단: 핵심 조언 (나무 심기 등)
3문단: 미래 비전
4문단: 자기긍정 메시지
```

## 3. 작성 톤과 스타일 가이드

### 3.1 전반적 톤
- **친근하면서도 전문적**
- **2인칭 존댓말** 기본 (단, 비판 섹션에서는 반말 혼용)
- **비유와 이미지 풍부**
- **직설적이되 따뜻함 유지**

### 3.2 문장 구성
- **짧고 강렬한 문장**과 **긴 설명 문장** 교차
- **질문형 도입** 자주 사용
- **"~네요", "~입니다", "~하세요"** 등 다양한 종결어미

### 3.3 비유 표현
- 자연물 (강, 산, 나무, 꽃, 흙, 물)
- 날씨/계절 (소나기, 폭풍, 한파, 무더위)
- 일상적 사물 (댐, 금고, 창고, 도끼)
- 동물 (장미, 황소, 양, 카멜레온)

### 3.4 강조 기법
- **"솔직히", "팩트폭격", "제발"** 등 강조 부사
- **느낌표 적극 활용**
- **괄호 안에 보충 설명** 빈번
- **질문 → 대답** 구조

## 5. 핵심 작성 원칙

### 원칙 1: 모든 해석은 사주 데이터에 근거
- 임의로 창작하지 않기
- 반드시 오행, 십성, 신살 등 근거 명시

### 원칙 2: 비유는 풍부하되 일관성 유지
- 한 사람의 사주를 하나의 자연 풍경으로 통일
- 예: 물 많은 사주 → 강/비/호수로 일관

### 원칙 3: 균형잡힌 시각
- 단점 → 장점으로 재해석하는 파트 반드시 포함
- 격려와 비판의 적절한 배분

### 원칙 4: 실용성
- "~하세요"로 끝나는 구체적 행동 지침
- 추상적 조언보다 실천 가능한 팁

### 원칙 5: 감성 터치
- 독자의 고난에 공감
- 격려와 희망의 메시지로 마무리

## 6. 핵심 개념: 신강/신약 판단 기준

### 신강(身强) - 일간이 강한 경우
**판단 기준:**
- 비견/겁재가 2개 이상 많음
- 인성(정인/편인)이 강하게 일간을 생해줌
- 월령(월지)을 얻음 (일간과 같은 오행이거나 생해주는 오행)
- 십이운성에서 제왕/건록/관대 등 왕성한 자리

**특징:**
- 자아가 강하고 독립적
- 리더십과 추진력 있음
- 자율성과 주도권 필요
- 경쟁에 강함

**직업 성향:**
- 창업가, 프리랜서, 독립 사업
- 의사결정 권한이 있는 리더 역할
- 도전적이고 역동적인 환경

---

### 신약(身弱) - 일간이 약한 경우
**판단 기준:**
- 재성/관성/식상이 과다하여 일간을 극하거나 설기함
- 비견/겁재와 인성이 약하거나 없음
- 월령을 잃음 (일간을 극하거나 설기하는 오행)
- 십이운성에서 절/태/사/병 등 약한 자리

**특징:**
- 주변 환경의 영향을 많이 받음
- 협업과 지원 시스템 필요
- 안정성과 체계 선호

- 조직 내 전문가, 팀 협업 역할

## 7. 자주 쓰이는 해석 공식

### 십성 결핍 관련
**무관 사주**
→ "자유로운 영혼, 조직 부적합, 고집, 자영업 추천"

**무재 사주**
→ "돈에 초연, 정신적 가치 중시, 안정적 직업 추천"

**무식 사주**
→ "표현력 약함, 내성적, 전문직 추천"

### 오행 불균형 관련
**재다신약**
→ "돈은 많이 들어오나 통제 어려움, 흐름 조심, 부동산 투자"

**신강재약**
→ "능력은 있으나 기회 적음, 적극성 필요"

**관왕신약**
→ "스트레스 많음, 책임감 과다, 휴식 필요"

### 형충파해합 관련
**원진/귀문관살**
→ "예민함, 직관력, 애증, 불안, 거리두기 필요"

**충(沖)**
→ "변동성, 불안정, 역동성, 이동수"

**형(刑)**
→ "자기 파괴적, 내적 갈등, 자제력 필요"

### 신살 관련
**도화살/홍염살**
→ "이성 인기, 매력, 충동성, 유혹 조심"

**역마살**
→ "이동수, 해외운, 역동적 직업"

**화개살**
→ "예술적 재능, 고독, 영적 성향"

**천을귀인/태극귀인**
→ "귀인복, 위기 탈출, 보호받음"

### 십성 과다 관련
**비견/겁재 많음**
→ "경쟁심, 자존심, 친구와 갈등, 동업 금지"

**식상 강함**
→ "아이디어형, 표현력, 예술, 자유업"

**인성 강함**
→ "공부 머리, 센스, 의존성, 현실 감각 부족"

**재성 강함**
→ "현실적, 돈 감각, 욕심, 배우자 고생"

**관성 강함**
→ "책임감, 스트레스, 규율, 권위"

### 배우자성 가시성 패턴

**천간에 배우자성**
→ "배우자 인연 명확, 일찍 결혼 고려, 관계 경로 분명"

**지지 주기운에 배우자성**
→ "배우자 인연 탄탄, 안정적 관계, 지속 가능성 높음"

**일지 지장간에만 배우자성**
→ "서서히 드러나는 인연, 친구→연인→배우자 발전, 시간 필요"

**다른 지장간에만 배우자성**
→ "약한 인연, 타이밍 중요, 적극적 만남 필요"

**배우자성 완전 부재**
→ "대운 의존적, 특정 시기 인연 활성화, 조급함 금물"

### 일간 강약별 관계 태도

**신강 + 관계**
→ "주도적, 통제 거부, 존중 요구, 독립성 중시"

**신약 + 관계**
→ "지지 추구, 안정감 필요, 적응적, 과도한 수용 주의"
