# 일간별 십성(十星) 매핑 완전판

## 사용법

사주 분석 시 일간(日干)을 먼저 확인한 후, 해당 테이블을 참조하여 나머지 천간과 지지의 십성을 빠르게 판단합니다.

**판단 순서:**
1. 일간 확인 (예: 甲木, 己土 등)
2. 해당 일간 테이블 찾기
3. 천간/지지의 오행을 십성으로 변환
4. 편(偏)과 정(正) 구분 주의

## 甲木(갑목) 일간

| 천간/지지 오행 | 십성 | 설명 |
|------------|------|------|
| **甲木** | 비견 (比肩) | 나와 같음, 동료, 경쟁자 |
| **乙木** | 겁재 (劫財) | 나와 비슷하나 음양 다름, 재물 빼앗김 |
| **丙火** | 식신 (食神) | 여유로운 표현, 창작 |
| **丁火** | 상관 (傷官) | 날카로운 표현, 반항 |
| **戊土** | 편재 (偏財) | 움직이는 재물, 사업 |
| **己土** | 정재 (正財) | 고정된 재물, 월급 |
| **庚金** | 편관 (偏官) | 강한 통제, 칠살 |
| **辛金** | 정관 (正官) | 정당한 권위, 직장 |
| **壬水** | 편인 (偏印) | 비전통적 학습, 효신 |
| **癸水** | 정인 (正印) | 정통 학습, 어머니 |

**지지 오행 → 십성 변환:**
- 인목(寅)/묘목(卯) → 비견/겁재 계열
- 사화(巳)/오화(午) → 식상 계열
- 진토(辰)/술토(戌)/축토(丑)/미토(未) → 재성 계열
- 신금(申)/유금(酉) → 관성 계열
- 해수(亥)/자수(子) → 인성 계열

## 편(偏)과 정(正) 구분 원칙

### 음양 법칙
- **같은 음양** (양-양, 음-음) → **비견**
- **다른 음양** (양-음, 음-양) → **겁재/식신/편재/편관/편인** 또는 **상관/정재/정관/정인**

### 구체적 규칙

**비견 vs 겁재:**
- 일간과 천간/지지가 **완전히 같음** → 비견
- 일간과 오행은 같지만 **음양 다름** → 겁재

**식신 vs 상관:**
- 일간이 생하는 오행이면서 **음양 같음** → 식신
- 일간이 생하는 오행이면서 **음양 다름** → 상관

**편재 vs 정재:**
- 일간이 극하는 오행이면서 **음양 같음** → 편재
- 일간이 극하는 오행이면서 **음양 다름** → 정재

**편관 vs 정관:**
- 일간을 극하는 오행이면서 **음양 같음** → 편관
- 일간을 극하는 오행이면서 **음양 다름** → 정관

**편인 vs 정인:**
- 일간을 생하는 오행이면서 **음양 같음** → 편인
- 일간을 생하는 오행이면서 **음양 다름** → 정인

## 실전 사용 예시

### Case 1: 甲木 일간 사주 분석

```
사주: 壬申 己未 甲寅 丙寅
일간: 甲木

십성 판단:
- 壬 (임수) → 편인 (偏印) - 甲은 양목, 壬은 양수, 음양 같음
- 申 (신금) → 편관 (偏官) - 신금은 양금
- 己 (기토) → 정재 (正財) - 기토는 음토, 음양 다름
- 未 (미토) → 정재 (正財) - 미토는 음토
- 甲 (갑목) → 비견 (比肩) - 일간과 동일
- 寅 (인목) → 비견/겁재 계열
- 丙 (병화) → 식신 (食神) - 병화는 양화, 음양 같음
- 寅 (인목) → 비견/겁재 계열

결론:
편인 1, 편관 1, 정재 2, 비견 3, 식신 1
→ 비견이 많은 신강 사주, 자영업/창업 추천
```

### Case 2: 己土 일간 사주 분석

```
사주: 丙子 癸巳 己未 壬申
일간: 己土

십성 판단:
- 丙 (병화) → 정인 (正印) - 병화는 양화, 음양 다름
- 子 (자수) → 정재 (正財) - 자수는 양수, 음양 다름
- 癸 (계수) → 편재 (偏財) - 계수는 음수, 음양 같음
- 巳 (사화) → 정인 (正印) - 사화는 음화
- 己 (기토) → 비견 (比肩) - 일간과 동일
- 未 (미토) → 비견 (比肩) - 미토는 음토
- 壬 (임수) → 정재 (正財) - 임수는 양수, 음양 다름
- 申 (신금) → 상관 (傷官) - 신금은 양금, 음양 다름

결론:
정인 2, 정재 3, 편재 1, 비견 2, 상관 1
→ 재다신약 구조, 재테크 중요
```

## 주의사항

1. **지장간 처리**: 지장간의 천간들도 같은 방식으로 십성 판단
2. **합화 고려**: 천간합이나 지지합으로 오행이 변했다면 변한 오행으로 판단
3. **십이운성 병행**: 십성과 함께 십이운성도 확인하여 세력 판단
4. **음양 실수 주의**: 편정 구분은 음양에 달려있으므로 정확히 구분
